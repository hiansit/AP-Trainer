# Absolute Pitch Trainer (絶対音感トレーニング)

ブラウザ上で動作する本格的な絶対音感トレーニングアプリです。
高品質なグランドピアノの音源を使用し、ゲーム感覚で耳を鍛えることができます。

## ✨ 特徴

- **本格的なピアノ音質**: Salamander Grand Pianoの高品質サンプルを使用。本物のピアノに近い音でトレーニングが可能です。
- **柔軟なトレーニング設定**: 1オクターブから4オクターブまで、レベルに合わせて音域を選択できます（1セット5問〜20問）。
- **リアルタイム・フィードバック**: 正解するとストリーク（連続正解）が記録され、不正解の場合は「どのくらい音が外れていたか（例：2半音高い）」を具体的にフィードバックします。
- **学習履歴データの保存**: IndexedDBを使用し、セッションごとの正解率や統計データをブラウザ内に自動保存。成長の推移をグラフで確認できます。また、不要な履歴の個別削除や一括削除、JSON形式でのデータダウンロード（エクスポート）にも対応しています。
- **フリープレイモード**: トレーニング中以外は、画面上のピアノ鍵盤を自由に演奏できる簡易ピアノとして機能します。
- **フルキーボード対応**: PC体験を最適化し、キーボード入力だけで演奏や回答のアクションが完結します。
- **マルチユーザー対応**: 複数のユーザーを登録し、それぞれの学習進捗と設定を切り替えて使用できます。

## 🎮 遊び方 / 使い方

### ▶️ はじめかた
1. ページを開き、初回はご自身のニックネームなどを入力してデータを識別できるようにします。
2. ホーム画面で「レベル」と「問題数」を選び、**「トレーニング開始」**（または `Enter` キー）を押します。

### 🎵 操作方法
画面上の鍵盤をクリック／タップするか、PCのキーボードを使用して演奏・回答ができます。

**キーボードマッピング (QWERTY配列):**
- **白鍵 (C〜B):** `A` `S` `D` `F` `G` `H` `J`
- **黒鍵 (C#〜A#):** `W` `E` `T` `Y` `U`
- **高いオクターブの白鍵:** `K` `L` `;`
- **オクターブの変更 (上下):** `Z` / `X` 
- **もう一度出題音を聴く:** `Space` または `R` 
- **トレーニングの中断:** `Escape` 

## 🛠️ 技術スタック

本アプリはビルドプロセスを必要としない Vanilla JavaScript で構築されており、ブラウザのみで動作します。

- **Frontend**: HTML5, Vanilla CSS, Vanilla JavaScript
- **Audio Engine**: [Tone.js](https://tonejs.github.io/) (Web Audio API ラッパー)
- **Database**: IndexedDB (ネイティブAPI)
- **UI Design**: CSS Glassmorphism & Custom Properties
